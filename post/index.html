If we’re careless in how we handle shared mutable state cause then it will cause problems. This is some random thoughts about how we deal with it. Git Git is a good example of a system to manage shared mutable state. It works well because there is almost no shared state mutation. It gives the appearance of having mutable state because “files do change.” There is no actual state mutation there though because a commit just creates a new object.
